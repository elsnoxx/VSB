# SWI - Úvod do softwarového inženýrství
Tento repositář je věnovaný předmětu SWI na VŠB FEI

## Zadání

```
Jako zápočet prosím vypracujte vlastní mini projekt, který bude obsahovat 5 typů diagramů se zaměřením na 
důslednost při používání správné notace:

    - Aktivitní diagram s alespoň 5 aktivitami, nějakou alternativou či
    - Sekvenční diagram s komunikací alespoň 5 zprávami, nějakou alternativou či loopem
    - Třídní diagram - alespoň 5 tříd
    - Use case diagram - nějaké provázání include či extends
    - Stavový diagram jednoho objektu s několika stavy
```

## Repositář

Diagramy jsou dělane za pomocí online nástoje [drav.io](https://app.diagrams.net/). Soubory, které jsou otevřitelné v tomto programu jsou v záložce drawio. Dále v založce jsou vygenerované tyto diagramy v pdf formě.
