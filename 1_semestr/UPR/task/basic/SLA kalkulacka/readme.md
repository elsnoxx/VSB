# SLA kalkulačka

## Zadání
- Program, který vypočte dostupnost služby za rok v procentech (např. 99,9). Pro kontrolu můžete použít např. [SLA Calculator](https://uptime.is/99.9)

## Překlad a souŠtění programu
- na překlad se používá překladač GCC, překlad a automatické spouště udělate za pomocí příkazu make
