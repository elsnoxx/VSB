# Guesing game

## Zadání
- Hra vygeneruje náhodné číslo. Uživatel bude opakovaně hádat číslo, program mu napíše, jestli je jeho tip moc velký nebo moc malý. Jakmile uživatel uhádne číslo, tak program skončí.

## Překlad a souŠtění programu
- na překlad se používá překladač GCC, překlad a automatické spouště udělate za pomocí příkazu make
