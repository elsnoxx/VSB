# Hod kostkou

## Zadání
- Program, který 10 000x hodí kostkou (vygeneruje náhodné číslo v rozsahu [1, 6]) a poté vypíše průměrnou hodnotu všech vygenerovaných hodnot.

## Překlad a souŠtění programu
- na překlad se používá překladač GCC, překlad a automatické spouště udělate za pomocí příkazu make 


## Ukázka
```c
make
```
