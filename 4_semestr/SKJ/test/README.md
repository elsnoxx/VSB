<h2>Testování programu</h2>
<p>K dispozici máte unit testy (<code>tests.py</code>), které ověřují chování vaší implementace a které lze
spustit pomocí knihovny <code>pytest</code> (ideálně uvnitř virtuálního prostředí).</p>
<pre class="highlight"><code class="language-bash hljs" data-highlighted="yes">$ python3 -m pip install pytest   <span class="hljs-comment"># instalace pytestu</span>
$ python3 -m pytest tests.py      <span class="hljs-comment"># spuštění unit testů</span></code></pre>
<p>To, že všechny testy prošly, však ještě neznamená, že je váš program správně :)
Zároveň pokud některé testy neprojdou, neznamená to ještě samo o sobě, že vaše řešení dostane <code>0</code>
bodů.</p>
