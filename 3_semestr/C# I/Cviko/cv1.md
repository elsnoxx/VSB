### Úkol na cvičení
- V prostředí Visual Studio založte nový projekt typu konzová aplikace.
- Upravte metodu Main, tak aby se program po spuštění zeptal na jméno uživatele. Poté co uživatel jméno zadá, vypište zpět do konzole vetu "Ahoj XYZ!". XYZ nahraďte jménem uživatele převedeným na velká písmena (uppercase).
- Aplikaci otestujte.
- Proveďte totéž v prostředí VS Code (příkaz [dotnet](https://learn.microsoft.com/en-us/dotnet/core/tools/dotnet))
- Seznamte se s adresářovou strukturou vámi založeného projektu.
