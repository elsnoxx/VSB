Main function that processes files and finds paths in a cave system. 

This function loads cave system data from text files and, for each file, finds all possible paths using the \doxylink{class_path_finder}{Path\+Finder}. The results are printed to the console.

\begin{DoxyReturn}{Returns}
Returns 0 if all files are successfully processed, or 1 if an error occurs. 
\end{DoxyReturn}
